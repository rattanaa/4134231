สร้าง Web App ระบบประปาหมู่บ้าน  รองรับการใช้งานทั้ง Mobile และ PC มีระบบ Login สำหรับผู้ใช้น้ำและผู้ดูแลระบบ พร้อมฟอร์มสมัครสมาชิกที่เก็บข้อมูล ชื่อ-นามสกุล, ที่อยู่, เบอร์โทร, เลขมิเตอร์น้ำ และรหัสผ่าน มีหน้าตรวจสอบค่าน้ำประจำเดือน, ประวัติการใช้น้ำ, แจ้งปัญหา, ข่าวประชาสัมพันธ์ และแก้ไขข้อมูลส่วนตัว ส่วนผู้ดูแลสามารถจัดการสมาชิก, บันทึกเลขมิเตอร์, คำนวณค่าน้ำอัตโนมัติ, ออกใบแจ้งหนี้, บันทึกการชำระเงิน, จัดการข่าวสาร และดูรายงานสรุป พร้อม UI ที่ทันสมัย ใช้งานง่าย


# ระบบบริหารจัดการการประปาหมู่บ้าน (Next.js)

โปรเจกต์นี้ใช้ Next.js เพื่อพัฒนา Web App สำหรับระบบประปาหมู่บ้าน รองรับทั้งการใช้งานบนมือถือและเดสก์ท็อป

## คุณสมบัติหลัก

- ระบบ Login สำหรับผู้ใช้น้ำและผู้ดูแลระบบ
- ฟอร์มสมัครสมาชิก พร้อมเก็บข้อมูล:
  - ชื่อ-นามสกุล
  - ที่อยู่
  - เบอร์โทรศัพท์
  - เลขมิเตอร์น้ำ
  - รหัสผ่าน
- หน้าตรวจสอบค่าน้ำประจำเดือน
- ประวัติการใช้น้ำ
- แจ้งปัญหา
- ข่าวประชาสัมพันธ์
- แก้ไขข้อมูลส่วนตัว
- สำหรับผู้ดูแลระบบ:
  - จัดการสมาชิก
  - บันทึกเลขมิเตอร์
  - คำนวณค่าน้ำอัตโนมัติ
  - ออกใบแจ้งหนี้
  - บันทึกการชำระเงิน
  - จัดการข่าวสาร
  - ดูรายงานสรุป

## ขั้นตอนเริ่มต้นสำหรับ Next.js

1. ติดตั้ง Node.js (แนะนำเวอร์ชัน LTS)
2. เปิด terminal ที่โฟลเดอร์ `LAB-1`
3. ติดตั้ง dependencies:
   ```bash
   npm install
   ```
4. รันโปรเจกต์ในโหมดพัฒนา:
   ```bash
   npm run dev
   ```
5. เปิดเบราว์เซอร์ไปที่:
   ```bash
   http://localhost:3000
   ```

## คำสั่งที่ใช้บ่อย

- `npm run dev` - รันแอปในโหมดพัฒนา
- `npm run build` - สร้างแอปสำหรับ production
- `npm start` - รันแอปในโหมด production
