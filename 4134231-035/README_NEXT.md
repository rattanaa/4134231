# Next.js scaffold for this repo

Follow these steps to run the development server locally:

1. Install dependencies:

```bash
npm install
```

2. Run the dev server:

```bash
npm run dev
```

Open http://localhost:3000

Notes:
- This adds a minimal Next.js app under `pages/` and a `package.json` with `next`, `react`, and `react-dom`.
- The main README files were left unchanged.
