# ระบบเว็บแอปตรวจเช็คค่าน้ำประปาหมู่บ้าน

เว็บแอปนี้ถูกออกแบบตามข้อกำหนดใน `SRS.md` เพื่อให้ผู้ใช้น้ำและผู้ดูแลระบบสามารถ:
- ลงชื่อเข้าใช้และจัดการบัญชีผู้ใช้
- ดูข้อมูลโปรไฟล์และที่อยู่
- ดูยอดค่าน้ำปัจจุบันและประวัติการใช้น้ำ
- ผู้ดูแลระบบบันทึกค่ามิเตอร์และสร้างรายงาน
- ใช้งานได้บนมือถือและ PC ด้วย UI แบบ Responsive

## โครงสร้างโปรเจกต์

- `architecture.md` - ออกแบบสถาปัตยกรรมระบบ
- `CLAUDE.md` - บันทึกข้อมูลหรือคำสั่งสำหรับ AI
- `readme.md` - สรุปโครงการและโครงสร้าง
- `SRS.md` - ข้อกำหนดซอฟต์แวร์
- `task.md` - งานและรายละเอียดที่ต้องทำ

### โฟลเดอร์ `backend/`

- `backend/next.config.js` - ตั้งค่า Next.js
- `backend/package.json` - dependencies และสคริปต์
- `backend/README.md` - เอกสารเฉพาะ backend
- `backend/pages/index.js` - หน้าแรกของเว็บ
- `backend/pages/api/auth.js` - API สำหรับยืนยันตัวตน
- `backend/pages/api/hello.js` - ตัวอย่าง endpoint
- `backend/pages/api/meters.js` - API จัดการค่ามิเตอร์
- `backend/pages/api/profile.js` - API ข้อมูลโปรไฟล์ผู้ใช้
- `backend/pages/api/reports.js` - API รายงาน

## ฟังก์ชันสำคัญตาม SRS

- ยืนยันตัวตนผู้ใช้
- ดู/แก้ไขโปรไฟล์และที่อยู่
- ดูประวัติการใช้น้ำและใบแจ้งหนี้
- ผู้ดูแลบันทึกค่ามิเตอร์และจัดการรายงาน
- มี UI ที่รองรับทั้งมือถือและพีซี
