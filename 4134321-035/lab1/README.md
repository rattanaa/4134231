# ระบบน้ำประปาหมู่บ้าน

เว็บแอปตัวอย่างสำหรับระบบน้ำประปาหมู่บ้านที่มีฟีเจอร์
- ลงชื่อเข้าใช้
- ดูข้อมูลผู้ใช้น้ำ
- อัปเดตที่อยู่ผู้ใช้
- ตรวจสอบสถานะการชำระเงิน

## วิธีเปิดใช้งาน

รันคำสั่งต่อไปนี้ในโฟลเดอร์นี้:

```bash
python -m http.server 8000
```

จากนั้นเปิดเบราว์เซอร์ที่:

```text
http://127.0.0.1:8000/
```
