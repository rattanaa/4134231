# Claude Project Notes

## Project Overview
ระบบเว็บแอปสำหรับจัดการข้อมูลผู้ใช้น้ำประปาหมู่บ้าน

## Features
- ลงชื่อเข้าใช้งาน
- แสดงข้อมูลผู้ใช้
- แก้ไขที่อยู่ผู้ใช้
- แสดงสถานะการชำระเงินและปริมาณน้ำ

## Tech Stack
- HTML
- CSS
- JavaScript
- LocalStorage สำหรับจัดเก็บข้อมูล

## Run Instructions
1. เปิดไฟล์ index.html ในเบราว์เซอร์
2. หรือรันคำสั่ง `python -m http.server 8000`
