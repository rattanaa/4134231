# Requirements

## Functional Requirements
- ระบบต้องมีหน้าลงชื่อเข้าใช้
- ระบบต้องแสดงข้อมูลผู้ใช้น้ำหลังจากเข้าสู่ระบบสำเร็จ
- ระบบต้องอนุญาตให้ผู้ใช้แก้ไขที่อยู่ของตนเอง
- ระบบต้องแสดงสถานะชำระค่าบริการและปริมาณน้ำ

## Technical Requirements
- ใช้ภาษา HTML, CSS, JavaScript
- ใช้ LocalStorage ในเบราว์เซอร์
- รองรับการใช้งานบนเบราว์เซอร์ทั่วไป
